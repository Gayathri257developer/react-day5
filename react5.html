4. What is the use of useCallback and useMemo

useMemo that allows to memoize expensive functions so that accepts a second parameter to declare dependencies. The expensive function will only run when its dependencies have changed.

useCallback will return a memoized version of the callback that only changes if one of the dependencies has changed. This is useful when passing callbacks to optimized child components that rely on reference equality to prevent unnecessary renders.

5. Why do we need keys in react list

Keys help React identify which items have changed (added/removed/re-ordered). To give a unique identity to every element inside the array, a key is required.
